# Sources techniques verifiees

Les fichiers Wokwi ont ete prepares a partir de la documentation officielle :

- Format `diagram.json` : https://docs.wokwi.com/de-DE/diagram-format
- Materiel supporte : https://docs.wokwi.com/getting-started/supported-hardware
- Simulation ESP32 : https://docs.wokwi.com/guides/esp32
- LED : https://docs.wokwi.com/parts/wokwi-led
- Resistance : https://docs.wokwi.com/parts/wokwi-resistor
- Servo : https://docs.wokwi.com/parts/wokwi-servo
- Potentiometre : https://docs.wokwi.com/parts/wokwi-potentiometer
- HC-SR04 : https://docs.wokwi.com/parts/wokwi-hc-sr04
- DHT22 : https://docs.wokwi.com/parts/wokwi-dht22
- Wi-Fi Wokwi : https://docs.wokwi.com/parts/wokwi-wifi-ap

Note importante : la simulation locale dans `simulateur-local/` ne depend pas de Wokwi ni d'un compte externe. Elle reste donc l'option offline et gratuite.
